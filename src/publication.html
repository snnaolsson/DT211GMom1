Arbetsprocessen

Jag börjar med att installera parcel.

Struktur

Hemsidan ligger i VS i en src-katalog. I den katalogen så finns underkataloger för bilder, css och js. Detta för att ha en lättöverskådlig struktur i arbetet. En gitignore-fil finns även i rotkatalogen och i den finns de delar som inte ska sparas till git.

Automatisering

Git

Sidan är kopplad till git och har ett github-repo. I repot finns två olika branches, en som heter main och en som heter develop. Under utvecklingsfasen av en ny del eller funktion på webbsidan så commitas alla förändringar till develp branchen. Detta för att inte publicera flera ofärdiga lösnignar till webbplatsen. När funktionen eller delen på sidan är klar och fungerar som den ska så mergas develop med main.

Publicering

Hemsidan publiceras automatiskt via Netlify. Hemsidan är kopplad till mitt repository på github och när "npm run build" körs så uppdateras automatiskt hemsidan utifrån dist-katalogen.
